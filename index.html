哈哈哈我又来日记了

昨天写了第一篇日记，感觉良好。这周要把Chevereto装到自己虚拟机里，所以用了之前安装的ubuntu16。之前都是用的centos7,命令也还记得一些，但这个ubuntu的命令又有许多不同，所以干脆写篇博客。

承载未来大画家的图展空间

本来是chevereto的，但是找到了更好的展示本画家艺术细胞的形式，3D图片展示。这个能在一个空间里展览你的图片，但不能放大图片。
参考博客：小坏孩

ubuntu16的网络配置

  1. 防火墙关闭
    sudo ufw disable
  2. 更改为静态ip
    sudo vi /etc/network/interfaces
    auto ens33
    iface ens33 inet static
    address 192.168.56.129
    gateway 192.168.56.2
    netmask 255.255.255.0
  3. 重启网络服务
    sudo /etc/init.d/networking restart

ubuntu16的openssh安装

如果不安装，xshell无法连接。
输入命令安装：
sudo apt-get install openssh-server
若没有ssh，需要执行
sudo apt-get install ssh

vi 命令的使用（大部分与vim相同）

  1. 删除
    未编辑状态按两次d，删除一行
    x删除光标前面，X删除光标后面

第一篇日记

发表于 2020-10-18

博客搭好却没写过什么

博客是九月十五号搭好的，搭好那天非常开心，也有尝试用MD格式来写写，熟悉了基本的格式，之前也有学习过一点。
后来就又在研究Sakura主题，看着别人搭建的博客，非常羡慕，就动手自己也尝试了一下，改各种配置，添加歌单，添加看板娘，添加评论，现在这个周末又过了，这周没有添加任何东西，可惜了。不过图集服务下周一定要实现。
#

漫无目的的浪费时间

自己目标不够明确，不知道自己需要什么，所以感觉对生活丧失了兴趣，又觉得生活枯燥了。用游戏打发时间，却在游戏结束后觉得索然无味，却无法停下来，放纵自己。
#

什么都想做却什么都没完成

学写博客

也没学什么东西，也写不出什么技术类的博客，所以就只能写写日记了，不过这也算是一个好开头吧，第一篇日记，这篇日记是五六年第一篇？记得上一次写日记是小学要交的100字的日记。有关日记还能想到在培训学校有个女生说的，如果想多了解自己，去写日记吧，那一次真的是个我最蠢的记忆，废话了一堆，把自己内心和过去完完全全的暴露给别人看，然后还觉得自己特别厉害，不同于常人。那种事情永远不要 再来一次 ?想想现在的状况似乎和那时大不一样了，明天把这个事写下来吧。
#

学画画

各种软件装了很久，也就画过一两次。想给朋友画个永恩，迟迟没行动。就像买了书却没去看，买了乐器却没去玩，但组装了电脑却一直在玩这一台，其它的只能闲置。

学PS

PS官网的教程国庆假学了一部分，算是入了个门，想着自己给自己处理结婚照的时候不用花钱还能为所欲为，就觉得非常爽，所以就学了个皮毛，还需把后面的教程学完。装了好几个版本的PS,各种破解安装换了好几个方式。我也没有收集软件的怪癖啊，肯定是虚荣心作祟。

总结

1 每天写篇博客，应该就花个半小时。
2 待办项
* PS学习
* 永恩的画
* 图库
